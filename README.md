### instuctions
````
    FRONT-END:
    1.) get inside code-exam folder
    2.) open terminal
    3.) run the following:
        - npm install
        - npm run serve

    BACKEND
    1.) get inside code-exam-api folder
    2.) copy .env.example and paste in current directory
    3.) rename COPIED .env.example file into .env
    4.) run the following:
        - composer install
        - php artisan key:gen
        - php artisan migrate
        - php artisan serve

    p.s. make sure to install necessary tools
````
